---
title: 'Connecter une boîte mail custom'
description: 'Comment connecter une nouvelle boîte email à HuntReach via SMTP/IMAP manuellement (tous fournisseurs)'
---

---

### Étape 1 : Accéder à la page de connexion des emails

1. Rendez-vous sur [cette page](https://app.huntreach.com/dashboard/settings/emails) de HuntReach dédiée à la connexion des emails.
2. Cliquez sur le bouton permettant de **connecter une nouvelle boîte email** et sélectionnez l’option **"Custom Email"** pour une configuration manuelle.

---

### Étape 2 : Préparer les informations nécessaires

Avant de remplir le formulaire, assurez-vous de disposer des informations suivantes pour votre adresse email :
|**Champs** |**Description** |
| ------------------ | ----------------------------------------------------------------------------------------- |
| **Email** | L’adresse email que vous souhaitez connecter. (Exemple : thomas@huntreach.net) |
| **Display Name** | Le nom d’affichage pour vos emails. (Exemple : Thomas Pecquet ou Thomas de HuntReach) |
| **Password** | Le mot de passe de l’adresse email ou un mot de passe d’application (si applicable). |
| **IMAP Host** | L’adresse du serveur IMAP de votre fournisseur. (Exemple : `imap.gmail.com`) |
| **IMAP Port** | Le port IMAP utilisé pour la connexion sécurisée. (Exemple : `993`) |
| **SMTP Host** | L’adresse du serveur SMTP de votre fournisseur. (Exemple : `smtp.gmail.com`) |
| **SMTP Port** | Le port SMTP utilisé pour l’envoi sécurisé. (Exemple : `465`) |
| **Warmup Enabled** | (Optionnel) Indiquez si vous souhaitez activer le warmup pour cette adresse. (TRUE/FALSE) |
| **Signature** | (Optionnel) Ajoutez une signature personnalisée ou laissez vide si non nécessaire. |

---

### Étape 3 : Remplir le formulaire sur HuntReach

1. Dans le formulaire de connexion, renseignez les champs requis :
   - **Email** : Entrez l’adresse email à connecter.
   - **Display Name** : Ajoutez un nom professionnel ou personnalisé qui apparaîtra pour vos destinataires.
   - **Password** : Saisissez le mot de passe ou le mot de passe d’application.
   - **IMAP Host** et **IMAP Port** : Fournissez les informations IMAP (exemple : `imap.gmail.com`, port `993`).
   - **SMTP Host** et **SMTP Port** : Renseignez les informations SMTP (exemple : `smtp.gmail.com`, port `465`).
   - (Optionnel) **Signature** : Ajoutez une signature pour vos emails si nécessaire.
2. Vérifiez que toutes les informations sont correctes avant de passer à l’étape suivante.

---

### Étape 4 : Connecter l’adresse email

1. Cliquez sur le bouton **"Connect Email"** pour valider la configuration.
2. Si tout est correctement configuré, votre adresse email sera connectée avec succès.

---

### Étape 5 : Résolution des problèmes courants

Si vous rencontrez des difficultés :

- **Erreur de connexion :** Vérifiez votre mot de passe ou les informations IMAP/SMTP fournies par votre fournisseur.
- **Activation SMTP ou IMAP :** Assurez-vous que l’authentification SMTP/IMAP est activée dans les paramètres de votre fournisseur d’email.
- **Warmup non activé :** Vérifiez que le warmup est configuré si vous souhaitez échauffer l’adresse.
